# Petshop Website

Este projeto consiste em um website desenvolvido para um Petshop. O objetivo principal é apresentar a marca, serviços e facilitar o contato com os clientes através de uma interface web moderna e responsiva.

## Funcionalidades

- **Navegação Responsiva**: O site possui um menu de navegação que se adapta a dispositivos móveis (mobile-friendly), permitindo abrir e fechar o menu através de um botão de alternância (toggle), conforme implementado no `script.js`.
- **Fechamento Automático**: Ao clicar em um link do menu em dispositivos móveis, o menu se fecha automaticamente para melhorar a experiência do usuário.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Manipulação do DOM para interatividade do menu)

## Como Executar

1.  Baixe os arquivos do projeto.
2.  Abra o arquivo `index.html` (não incluído neste contexto, mas assumido como ponto de entrada) em qualquer navegador web moderno.

## Estrutura de Arquivos

- `script.js`: Contém a lógica para o funcionamento do menu responsivo.
